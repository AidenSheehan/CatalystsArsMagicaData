**Targets**
- Players can select actors as targets
- Comparison tests against actors
- Attacks on actors

**Items**
- In addition to actors, items can also be added to the site
- Like actors, items can be displayed or hidden via the SceneActorsSelector.
- The number of items can be customized
- Items can be locked or unlocked so that players can take them.

**Actor names**
- Players can display shared player names via mouseover.
