**Ziele**
- Spieler können Akteure als Ziel auswählen
- Vergleichsproben gegen Akteure
- Angriffe auf Akteure

**Items**
- Neben Akteuren können auhc Items der Lsite hinzugefügt werden
- Items können wie Akteuere über den SceneActorsSelector angezeigt bzw ausgeblendet werden.
- Anzahl der Items kann angepasst werden
- Items können gesperrt oder für freigegben werden, sodass Spieler sie sich nehmen können.

**Akteur-Namen**
- Spieler können freigegebene Akteur-Namen via Mouseover anzeigen.
