Funktionen des Moduls

Szene aktivieren

Über den Button im Figuren-Menü (links) können die SceneActors vom Spielleiter für die aktuelle Szene aktiviert bzw. deaktiviert werden. Spieler können die SceneActors über den Button temporör ausblenden. Sobald der Spielleiter einen neuen Akteur anzeigt, wird auch das Ausblenden beendet.

Akteure einer Szene auf einen Blick

Mit dem Akteur-Selektor des Moduls hast du alle für eine Szene relevante Akteure im Blick mit folgenden Möglichkeiten:

  • Du kannst Akteure direkt auf den Selektor ziehen, um sie hinzuzufügen.

  • Über das rechte Icon kannst du den Akteur wieder aus der Szenen-Liste entfernen.

  • Den Charakterbogen eines Akteurs anzeigen.

  • Den Akteur auf der Spielfläche anzeigen.

  • Den Akteur in den Fokus setzen. Er ist dadurch größer als alle anderen Akteure.

    Beispiel: Die Spieler sind im Gespräch mit diesem Akteur.

  • Über den Avatar in der Liste kannst du ebenfalls mit einem Klick den Akteur auf der Spielfläche anzeigen oder ausblenden oder mit einem Rechtsklick den Charakterbogen öffnen.

  • Für Spieler-Charaktere und NSCs gibt es jeweils eine eigene Liste. Während die NSC-Liste für jede Szene individuell ist, gilt die Spieler-Liste für die komplette Welt.

  • Beide Listen können einzeln ein- und ausgeblendet werden. Das betrifft aber nicht die Avatar-Anzeige auf der Spielfläche.

  • Die Größe der Avatare auf er Spielfläche kann verkleinert bzw. vergrößert werden.

  • Die Akteure können für die Spieler und den Spielleiter ausgeblendet werden.

Szenen-Einstellungen

Über das Zahnrad rechts in der Titelleiste werden die Szenen-Einstellungen geöffnet. Aktuell gibt es folgende Optionen:

  • Es können alle Akteure aus der Szene auf einmal entfernt werden.

  • Neben dem Zahnrad-Icon kann der Selektor minimiert und wieder maximiert werden.

Modul-Einstellungen

Hier finden sich globale oder Spielerbezogene Einstellungen.

  • SceneActors deaktivieren: Das betrifft den jeweils eingeloggten Spieler und sollte mit dem Spielleiter abgesprochen werden. Dadurch wird die Option, Akteure auf der Spielfläche anzuzeigen, für den Spieler komplett deaktiviert. Wichtig: Die Option deaktiviert nur die Anzeige, nicht aber das Modul an sich.

  • Anfänglich sind die Tooltips eine gute Hilfe, um sich zurechtzufinden. Später können sie ggf. störend sein. Hier können sie deaktiviert werden.

  • Eine nützliche Funktion beim Spielen mit einem digitalen Tabletop, z.B. einem liegenden TV: Die Akteure auf der Spielfläche können gedreht werden.

  • Die Anzeige des Changelogs nach einem Update kann deaktiviert werden.
